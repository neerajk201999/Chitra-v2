# Source-assisted reconstruction provenance benchmark

**Verified:** 2026-07-16 · **ADR:** 0023

The generated, non-copyrighted fixture proves:

- a licensed Intake reference is planned through Direction and Storyboard and
  cited by a source-assisted Score asset;
- a project-local image nested inside a figure renders in a real browser;
- editing only that nested image changes the scene hash;
- two captures at the same frame are byte-identical;
- the public `chitra creative-check` accepts the complete four-artifact chain;
- changing rights to `reference-only` blocks rendered use; and
- relabelling the same rendered reference bytes as `clean-room` is blocked.

This validates declared provenance and deterministic byte dependencies. It does
not independently establish legal rights, prove that authored HTML contains no
manually copied vector paths, or establish visual superiority. The user remains
the authority for `owned` and `licensed` claims.
